# Minishell
### _Собственная оболочка/bash на С_
- *Для компиляции и запуска*
```sh
make && ./minishell
```
- *Удаление объектных файлов и перекомпиляция*
```sh
make fclean // make re
```
*Основные команды bash и краткая шпаргалка по оболочке*
